Front Web Developer

Frontend web developer

Full Time
Facebook Inc.
No.14, Hope Rd, Ibadan, Nigeria

We are seeking a talented Frontend Web Developer with expertise in HTML, CSS, JavaScript, and ReactJS. You’ll play a key role in creating visually stunning and user-friendly interfaces. Your primary responsibilities will include translating design wireframes into code, building reusable components, and optimizing applications for maximum performance and scalability.Responsibilities:Develop responsive and interactive web pages using HTML, CSS, and JavaScript.Implement and optimize UI components with ReactJS to deliver a fast, seamless user experience.Collaborate with the design and backend teams to ensure consistency and functionality across the website.Maintain code quality, organization, and versioning using Git.Debug and troubleshoot cross-browser issues to ensure consistent rendering across all major browsers.Stay up-to-date with the latest trends in frontend development and web design best practices.Qualifications:Proven experience in frontend development with a focus on HTML, CSS, JavaScript, and ReactJS.Strong understanding of responsive design principles and web performance optimization.Experience with RESTful APIs, JSON, and asynchronous request handling.Familiarity with version control systems, particularly Git.A keen eye for design and attention to detail.Problem-solving skills and ability to work in a collaborative team environment.Nice to Have:Experience with modern frontend workflows, tools, and libraries (e.g., Webpack, Babel, Sass).Knowledge of frontend testing frameworks (e.g., Jest, Cypress).Familiarity with SEO best practices and principles.Why Join Us?Opportunity to work with a dynamic team on exciting projects.Flexible working hours with options for remote work.Competitive salary and growth opportunities within the company. A creative and supportive work environment. If you’re passionate about frontend development and want to be part of a team creating a next-generation online retail experience, we'd love to hear from you!

How to Apply:

Please send your resume, portfolio, and a brief cover letter. Be sure to highlight projects that demonstrate your skills in HTML, CSS, JavaScript, and ReactJS.
